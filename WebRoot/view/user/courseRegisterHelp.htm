Course Registration Help
Course Registration enables a user to register for any course in the Sample Run-Time Environment.

Registration Instructions
  1. Check the checkbox next to any of the courses you would like to register for. If you are already registered for any courses and wish to be removed from them, simply uncheck them. If you are removed from a course all of your data associated with that course will be removed.
  2. Click the Submit Form button.
